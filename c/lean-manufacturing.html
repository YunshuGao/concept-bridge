精益生产 jīngyì shēngchǎnlean manufacturing (the Toyota Production System)

🔊 lean manufacturing /lin ˌmænjəˈfækʧərɪŋ/

丰田开创的生产哲学:系统性消除一切不增值的浪费（muda）——过度生产、等待、搬运、库存、多余动作、缺陷、过度加工。核心是准时制（JIT）、持续改善（kaizen）与对流程的尊重。它重塑了全球制造业。

Toyota's production philosophy: systematically eliminate all non-value-adding waste (muda) — overproduction, waiting, transport, inventory, motion, defects, over-processing. Its core is just-in-time (JIT), continuous improvement (kaizen) and respect for the process. It reshaped global manufacturing.

Etymology 词源

lean（精瘦的）——MIT 学者1980年代为丰田体系命名,喻'去除一切赘肉的生产';muda（无駄）日语'浪费'。

Synonyms 近义词

  • ▸ the Toyota Production System /ðə tɔɪˈoʊtə pərˈdəkʃən ˈsɪstəm/
    The Toyota Production System turned waste elimination into a competitive weapon.
  • ▸ just-in-time /ʤɪst-ɪn-taɪm/
    Just-in-time makes inventory arrive exactly when needed, and never sooner.

Nuance 陷阱与辨析

精益的准时制（JIT）以极低库存换效率,但2020–22年供应链危机暴露其脆弱:一环断裂,全线停摆——效率与韧性的经典权衡。
