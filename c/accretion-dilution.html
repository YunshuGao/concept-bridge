增厚与稀释 zēnghòu yǔ xīshìaccretion / dilution

🔊 accretion dilution /əˈkriːʃən dɪˈluʃən/

衡量一笔并购对收购方每股盈利（EPS）的即时影响:若交易后 EPS 上升即“增厚”，下降即“稀释”。它是并购谈判的快速财务检验，但只看短期 EPS、不看长期价值创造，易误导。

Measuring a deal's immediate effect on the acquirer's earnings per share: if EPS rises post-deal it's accretive, if it falls it's dilutive. A quick financial screen in deal negotiation — but one that looks only at near-term EPS, not long-term value creation, and so can mislead.

Etymology 词源

accrete 源自拉丁 accrescere（增长、附加）——增厚即“每股盈利被添加”；dilute 源自 diluere（冲淡）。

Synonyms 近义词

  • ▸ EPS impact /ˌiːpiːˈɛs ˌɪmˈpækt/
    The EPS impact is what the market reacts to first and forgives last.
  • ▸ earnings accretive /ˈərnɪŋz əˈkriːtɪv/
    An earnings-accretive deal can still destroy value if it was overpaid for.

In the wild 语境例句

A deal can be EPS-accretive simply by using cheap debt, yet destroy value — accretion is a financing artefact, not proof of a good acquisition.

Nuance 陷阱与辨析

增厚≠创造价值:用低息债务收购几乎必然增厚 EPS，但这只是杠杆的会计效果，与是否物有所值无关——是并购分析的经典陷阱。
