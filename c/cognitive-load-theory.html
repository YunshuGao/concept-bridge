认知负荷理论 rènzhī fùhè lǐlùncognitive load theory

🔊 cognitive load theory /ˈkɑgnɪtɪv loʊd ˈθɪri/

斯威勒:工作记忆容量有限,一次能处理的新信息很少。教学须管理三类负荷:内在负荷（内容本身的难度）、外在负荷（拙劣设计造成的额外负担,须削减）、相关负荷（用于建构图式的有效努力,须促进）。它为显性教学、示例、分步与去冗余提供了认知科学依据。

Sweller's theory: working memory is limited and can handle little new information at once. Teaching must manage three loads: intrinsic (the inherent difficulty of the content), extraneous (extra burden from poor design, to be cut), and germane (the useful effort of building schemas, to be fostered). It gives cognitive-science grounding to explicit teaching, worked examples, step-by-step sequencing and removing redundancy.

Etymology 词源

cognitive 源自拉丁 cognoscere（认知）;load（负荷）即工作记忆承受的信息量;Sweller 1980年代创立。

Synonyms 近义词

  • ▸ working memory limits /ˈwərkɪŋ ˈmɛməri ˈlɪmɪts/
    Working memory limits are why a novice drowns in a task an expert finds trivial.
  • ▸ the expertise reversal effect /ðə ˌɛkspərˈtiz rɪˈvərsəl ˈifɛkt/
    The expertise reversal effect means support that helps novices can hinder experts.

Collocations 常用搭配

In the wild 语境例句

Cognitive load theory says teaching syntax and logic and a new IDE all at once overloads novices — introduce one at a time so working memory can cope.
