夏普比率 xiàpǔ bǐlǜthe Sharpe ratio

🔊 Sharpe ratio /ʃɑːrp ˈreɪʃiˌoʊ/

衡量投资'每承担一单位风险获得多少超额回报'的指标:（回报−无风险利率）÷波动率。它让高回报但高波动的策略与稳健策略可公平比较——高回报若靠高风险堆出，Sharpe 比率会揭穿。

A measure of how much excess return an investment earns per unit of risk taken: (return − risk-free rate) ÷ volatility. It lets a high-return but volatile strategy be compared fairly with a steady one — if the return was bought with reckless risk, the Sharpe ratio exposes it.

Etymology 词源

以诺奖得主 William Sharpe 命名——把'回报'与'风险'压缩成一个可比数字，是绩效评估的通用语。

Synonyms 近义词

  • ▸ risk-adjusted return /rɪsk-əˈʤəstɪd rɪˈtərn/
    Risk-adjusted return is the only honest way to rank a cautious fund against a wild one.
  • ▸ return per unit of risk /rɪˈtərn pər ˈjunɪt əv rɪsk/
    Return per unit of risk asks not how much you made but how much you risked to make it.

Collocations 常用搭配

  • ▸ a high Sharpe ratio
  • ▸ risk-adjusted return
  • ▸ return per unit of volatility
  • ▸ beat the benchmark on a risk-adjusted basis

In the wild 语境例句

A hedge fund boasting 30% returns can have a worse Sharpe ratio than a steady 8% fund — the eye-catching number hid enormous risk.
