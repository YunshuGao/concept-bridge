有效前沿 yǒuxiào qiányánthe efficient frontier

🔊 efficient frontier /ɪˈfɪʃənt frənˈtɪr/

马科维茨投资组合理论的核心图像:在风险-回报平面上,由'给定风险下回报最高、或给定回报下风险最低'的所有最优组合连成的曲线。理性投资者只应持有前沿上的组合;前沿之下的组合都是'次优'(可在不加风险下提高回报)。资产配置的理论标尺。

The core image of Markowitz's portfolio theory: on the risk-return plane, the curve of all optimal portfolios that offer the highest return for a given risk (or the lowest risk for a given return). A rational investor should hold only portfolios on the frontier; anything below it is sub-optimal (you could raise return without adding risk). The theoretical yardstick of asset allocation.

Etymology 词源

frontier（边疆、前沿,拉丁 frons 前额）——'有效前沿'即最优组合所在的'边界曲线',其外不可达、其内不理性。

Synonyms 近义词

  • ▸ the optimal portfolio set /ðə ˈɑptɪməl pɔrtˈfoʊliˌoʊ sɛt/
    The optimal portfolio set is where you can't get more return without more risk.
  • ▸ mean-variance optimisation /min-ˈvɛriəns ɑptɪmɪˈzeɪʃən/
    Mean-variance optimisation searches for the frontier by trading expected return against variance.

In the wild 语境例句

Combining low-correlation assets pushes the efficient frontier up and to the left — more return for the same risk, the free lunch of diversification.

Nuance 陷阱与辨析

前沿依赖对回报、方差、相关性的估计,而这些极难准确估计(估计误差→'优化即放大误差');故实务常用更稳健的方法。理论优美,落地须谨慎。
