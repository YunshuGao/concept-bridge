价值链 jiàzhíliànthe value chain

🔊 value chain /ˈvælju ʧeɪn/

波特把企业拆成一串增值活动:进料物流、生产、发货、营销、服务（主要活动）+ 采购、技术、人力、基础设施（支持活动）。竞争优势来自某些环节比对手做得更便宜或更好。

Porter decomposes a firm into value-adding activities: inbound logistics, operations, outbound logistics, marketing, service (primary) plus procurement, technology, HR and infrastructure (support). Advantage lives in doing some links cheaper or better than rivals.

Etymology 词源

chain 源自拉丁 catena（锁链）——价值链即环环相扣的增值活动；每一环都可能是优势或漏点。

Synonyms 近义词

  • ▸ activity system /ækˈtɪvɪti ˈsɪstəm/
    An activity system wins when the links reinforce each other, not just individually.
  • ▸ value system /ˈvælju ˈsɪstəm/
    The value system extends the chain upstream and downstream, supplier to buyer.

Collocations 常用搭配

  • ▸ move up the value chain
  • ▸ unbundle the chain

In the wild 语境例句

Ask of every activity: does this create value the customer pays for, or just cost? The value chain forces that audit link by link.
