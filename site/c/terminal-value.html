终值 zhōngzhíterminal value

🔊 terminal value /ˈtərmənəl ˈvælju/

DCF 中，明确预测期结束后企业剩余全部年份价值的估算——通常用永续增长模型（戈登公式）或退出倍数法。终值常占总估值的大头，也是最脆弱的假设:一个永续增长率的微调即撬动整体估值。

In a DCF, the estimate of all the value beyond the explicit forecast period — usually via a perpetuity-growth (Gordon) model or an exit multiple. Terminal value often dominates the total and is the most fragile assumption: a small tweak to the perpetual growth rate swings the whole valuation.

Etymology 词源

terminal 源自拉丁 terminus（边界、终点）——终值即“预测边界之外的全部价值”。

Synonyms 近义词

  • ▸ the continuing value /ðə kənˈtɪnjuɪŋ ˈvælju/
    The continuing value assumes the business lives, and grows, forever.
  • ▸ the Gordon growth model /ðə ˈgɔrdən groʊθ ˈmɑdəl/
    The Gordon growth model values a perpetuity growing at a steady rate.

Collocations 常用搭配

  • ▸ the terminal value dominates
  • ▸ perpetuity growth rate
  • ▸ exit multiple method
  • ▸ the fade period

In the wild 语境例句

When terminal value is 75% of a DCF, the 'analysis' is really one bet on the perpetual growth rate dressed up as a spreadsheet.
