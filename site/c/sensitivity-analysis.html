敏感性分析 mǐngǎnxìng fēnxīsensitivity analysis

🔊 sensitivity analysis /ˌsɛnsɪˈtɪvɪti æˈnælɪsɪs/

逐一改变模型的关键假设（增长率、利润率、贴现率），观察结论如何变化——找出哪些变量真正驱动结果、结论有多脆弱。配合情景分析与蒙特卡洛模拟，把“一个数字”变成“一个范围与概率”。

Varying a model's key assumptions one at a time (growth, margin, discount rate) to see how the conclusion moves — identifying which variables really drive the result and how fragile it is. Paired with scenario analysis and Monte Carlo simulation, it turns 'one number' into 'a range with probabilities'.

Etymology 词源

sensitivity 源自拉丁 sentire（感觉）——敏感性即“结果对假设变化的感觉有多灵敏”。

Synonyms 近义词

  • ▸ what-if analysis /wət-ɪf æˈnælɪsɪs/
    What-if analysis finds the assumption the whole case secretly rests on.
  • ▸ the tornado chart /ðə tɔrˈneɪˌdoʊ ʧɑrt/
    A tornado chart ranks assumptions by how violently they swing the answer.

Collocations 常用搭配

  • ▸ stress the key assumptions

  • ▸ which variable drives the result
  • ▸ a football-field of values

In the wild 语境例句

Sensitivity analysis on a DCF reveals the valuation swings 40% on the discount rate alone — flagging the one number to debate hardest.
