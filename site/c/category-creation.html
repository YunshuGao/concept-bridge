品类创造 pǐnlèi chuàngzàocategory creation

🔊 category creation /ˈkætəˌgɔri kriˈeɪʃən/

不在既有品类里争第一，而是定义一个新品类并成为其代名词（如 CRM、云计算、电动车高端市场）。品类王往往攫取该品类绝大部分经济价值——先教育市场，再统治市场。

Rather than compete for first place in an existing category, define a new one and become synonymous with it (CRM, cloud, premium EVs). The category king typically captures most of the category's economic value — educate the market, then rule it.

Etymology 词源

category 源自希腊 katēgoria（陈述、指控）——品类即“把事物归为一类的陈述”；创造品类即发明一种新的归类方式。

Synonyms 近义词

  • ▸ category king /ˈkætəˌgɔri kɪŋ/
    The category king writes the definition everyone else must answer to.
  • ▸ point of view /pɔɪnt əv vju/
    Category creation sells a point of view before it sells a product.

Collocations 常用搭配

  • ▸ create a category
  • ▸ own the category
  • ▸ the category king
  • ▸ name the problem

In the wild 语境例句

Instead of selling 'better sales software', a firm names a new category and becomes the default answer when buyers ask for it.
