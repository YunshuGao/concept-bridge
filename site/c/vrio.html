VRIO 框架 VRIO kuàngjiàthe VRIO framework

🔊 VRIO /ˈvriːoʊ/

巴尼的资源筛选工具:一项资源要构成持续优势，须同时满足价值（Value）、稀缺（Rarity）、难以模仿（Inimitability）、组织能利用（Organisation）。四项俱全才是可持续优势的来源。

Barney's screen for resources: to yield sustained advantage, a resource must be Valuable, Rare, costly to Imitate, and the firm must be Organised to exploit it. Only when all four hold is the advantage durable.

Etymology 词源

VRIO 为 Value/Rarity/Imitability/Organisation 首字母缩略——咨询与商学院最爱的记忆锚，一词概括 RBV 的检验清单。

Synonyms 近义词

  • ▸ resource audit /ˈrisɔrs ˈɔdɪt/
    A resource audit runs each asset through VRIO and keeps only the survivors.
  • ▸ inimitability test /ɪnˌɪmɪtəˈbɪlɪti tɛst/
    The inimitability test asks: could a rival copy this cheaply? If yes, discard it.

Collocations 常用搭配

  • ▸ pass the VRIO test
  • ▸ valuable and rare
  • ▸ costly to imitate
  • ▸ organised to capture value

In the wild 语境例句

A great brand is valuable and rare, but if the firm isn't organised to exploit it — the O in VRIO — the advantage stays on paper.
