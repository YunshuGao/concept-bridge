敏感性表格 mǐngǎnxìng biǎogésensitivity (data) tables

🔊 sensitivity tables /ˌsɛnsɪˈtɪvɪti ˈteɪbəlz/

在估值模型中用Excel数据表把两个关键假设(如WACC与永续增长率、进入与退出倍数)做成二维网格,展示估值在各组合下的取值区间。它承认'单一数字是假象',把DCF从一个点变成一个范围,是估值汇报的标配。配合'足球场图'展示最终区间。

In a valuation model, an Excel data table arranges two key assumptions (WACC and perpetual growth, or entry and exit multiple) into a two-dimensional grid showing the valuation across every combination. It admits that a single number is an illusion, turning a DCF from a point into a range — standard in valuation output, paired with a 'football field' to display the final range.
