回归分析 huíguī fēnxīregression analysis

🔊 regression analysis /rəˈgrɛʃən æˈnælɪsɪs/

用一个方程刻画因变量与一个或多个自变量的关系(y=α+βx+ε),估计系数、拟合优度(R²)与显著性。金融中无处不在:估β(CAPM)、因子模型、预测、检验关系。它是把'相关'量化并逼近'因果'的主力工具——但相关不等于因果。

Modelling the relationship between a dependent variable and one or more independent variables with an equation (y = α + βx + ε), estimating coefficients, goodness of fit (R²) and significance. Ubiquitous in finance: estimating beta (CAPM), factor models, forecasting, testing relationships. The workhorse for quantifying association and approaching causation — though correlation is not causation.

Etymology 词源

regression 源自高尔顿'向均值回归'(regression to the mean)的研究——统计学借其名;拉丁 regredi（退回）。

Synonyms 近义词

  • ▸ linear regression /ˈlɪniər rəˈgrɛʃən/
    Linear regression fits the best straight line through a cloud of points.
  • ▸ the factor model /ðə ˈfæktər ˈmɑdəl/
    A factor model is a regression of returns on the drivers that explain them.

Collocations 常用搭配

  • ▸ run a regression
  • ▸ the beta coefficient
  • ▸ R-squared
  • ▸ multiple regression

In the wild 语境例句

Regressing a stock's returns on the market's returns yields its beta — the slope that CAPM (which the CFA leans on) then prices.

Nuance 陷阱与辨析

回归的经典陷阱:相关≠因果;过拟合;遗漏变量;共线性。R²高不代表模型对,可能只是曲线拟合(见 model-risk)。
